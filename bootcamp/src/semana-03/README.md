# Semana 03 – Diagrama UML del Dominio (Farmacia SaludPlus)

En esta semana se realizó la representación del dominio de la farmacia SaludPlus
mediante un diagrama de clases UML. Este diagrama refleja los objetos principales
del sistema, sus atributos, métodos y las relaciones entre ellos.

## Objetos del dominio:
- Producto
- Cliente
- Empleado
- Venta
- Inventario

## Actividades realizadas:
✔ Identificación de clases  
✔ Definición de atributos y métodos  
✔ Análisis de relaciones entre objetos  
✔ Elaboración del diagrama UML de clases

El diagrama permitirá avanzar a la implementación de clases en semanas posteriores,
sirviendo como base para el diseño orientado a objetos del sistema.
