# Semana 02 – Modelado del Dominio (Farmacia SaludPlus)

En esta semana se realizó el análisis del dominio asignado: la farmacia SaludPlus.
Se identificaron los objetos principales del sistema, junto con sus atributos, métodos
y relaciones.

Este modelado servirá de base para las semanas siguientes, incluyendo UML, el diseño
de clases y la implementación en Java.

## Objetos identificados:
- Producto
- Cliente
- Empleado
- Venta
- Inventario

## Entregables:
- Modelado de objetos
- Atributos y métodos
- Relaciones del dominio
- Clases simples del dominio en Java
